mod_lfo
---

## Description

A [midiglue](https://sigboost.audio/midiglue/) patch that drives MIDI modulation wheel messages with an LFO.

<img src="ss.png" width="500px">

## Control

* Knob L: LFO Rate 0.10 - 2.00 Hz
* Knob R: LFO Depth 0 - 100 %
* Button A: LFO ON
* Button B: LFO OFF
